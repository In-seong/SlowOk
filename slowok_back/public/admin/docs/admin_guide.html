CHAPTER 1
역할 및 권한 체계
시스템에는 MASTER와 ADMIN 두 가지 관리자 역할이 있습니다.

★ MASTER (최고 관리자)

  • ✓ 전체 기관 생성/관리
  • ✓ 기관 전환 (드롭다운)
  • ✓ ADMIN 계정 생성/삭제
  • ✓ 권한 설정 (ON/OFF)
  • ✓ 초대코드 발급
  • ✓ 플랜/기능 관리
  • ✓ 모든 ADMIN 기능 포함

■ ADMIN (기관 관리자)

  • ✓ 소속 기관 데이터만 접근
  • ✓ 사용자 관리/메모
  • ✓ 콘텐츠/챌린지 등록
  • ✓ 진단 검사 관리
  • ✓ 콘텐츠 할당/배정
  • ✓ 대시보드/리포트 조회
  • ✗ 기관 생성, 관리자 관리 불가
💡 TIP

ADMIN의 권한은 MASTER가 세밀하게 ON/OFF할 수 있습니다. 예: 대시보드만 볼 수 있는 조회 전용 ADMIN 생성 가능.

CHAPTER 2
초기 설정
서비스 사용을 위한 필수 설정 단계입니다. (MASTER 전용)
1

기관 생성 및 초대코드 발급

사이드바 > 운영 > 기관 관리
"기관 추가" 버튼 클릭 > 기관명, 유형, 주소 입력 > "자동 생성"으로 초대코드 발급. 이 코드를 학부모에게 전달합니다.

2

ADMIN 계정 생성

사이드바 > 시스템 > 관리자 관리
"관리자 추가" 버튼 > 아이디, 비밀번호, 이름, 역할(ADMIN), 소속 기관 선택. 생성된 계정 정보를 담당자에게 전달합니다.

3

권한 설정

사이드바 > 시스템 > 권한 관리
각 ADMIN별로 대시보드, 콘텐츠, 진단, 사용자 관리 등 기능 단위 ON/OFF. 변경 즉시 적용됩니다.

⚠ 주의사항

초대코드는 기관당 1개입니다. 변경 시 기존 코드는 즉시 비활성화되므로, 이미 전달한 코드가 있다면 사용자에게 안내하세요.

CHAPTER 3
대시보드 활용
로그인 후 첫 화면. 기관의 핵심 현황을 한눈에 파악합니다.
사이드바 > 대시보드
📊
요약 카드 (4개)

전체 사용자, 학습자, 학습 콘텐츠 수, 진단 완료 건수를 숫자로 표시합니다.

📈
학습 완료율

도넛 차트로 전체 배정 중 완료 비율을 퍼센트로 보여줍니다.

📊
7일 활동 추이

최근 7일간 일별 진단/학습완료/챌린지 활동을 바 차트로 표시합니다.

📈
주간 학습 추이 (8주)

라인 차트로 주별 학습 시작/완료 건수 추세를 확인합니다.

📈
월간 진단 현황 (6개월)

월별 진단 검사 수와 평균 점수를 바+라벨로 표시합니다.

📝
최근 진단/가입자

최근 5건의 진단 결과와 최근 가입자를 실시간으로 확인합니다.

💡 TIP

MASTER는 상단 기관 드롭다운으로 기관을 전환하면 해당 기관의 대시보드를 볼 수 있습니다. "전체"를 선택하면 통합 현황을 확인합니다.

CHAPTER 4
콘텐츠 관리
학습 카테고리, 콘텐츠, 챌린지를 등록하고 관리합니다.
1

카테고리 생성

콘텐츠 > 카테고리 관리
학습 영역(언어, 수학, 사회성 등)을 카테고리로 생성합니다. 아이콘과 설명을 추가할 수 있습니다.

2

학습 콘텐츠 등록

콘텐츠 > 콘텐츠 관리
카테고리 선택 > 제목, 유형(영상/퀴즈/게임/읽기), 난이도 설정 > 파일 업로드. 영상, 이미지, 음성 파일을 첨부할 수 있습니다.

3

챌린지 & 문항 등록

콘텐츠 > 챌린지 관리
챌린지 생성 후 문항(객관식, 주관식)을 추가합니다. 이미지 첨부, 정답, 순서 설정이 가능합니다.

4

보상 카드 등록

콘텐츠 > 보상카드 관리
챌린지 통과 시 지급할 보상 카드를 등록합니다. 이름, 설명, 이미지, 희귀도를 설정합니다.

CHAPTER 5
진단 검사 관리
학습자의 수준을 파악하기 위한 진단 검사를 생성하고 결과를 확인합니다.

진단 생성

진단 > 진단 관리
1

진단 테스트 생성

제목, 설명, 카테고리, 시간제한 설정

2

문항 추가

객관식/주관식, 보기, 정답, 순서 설정

3

학습자에게 배정

콘텐츠 할당에서 진단 선택 후 배정

결과 확인

진단 > 진단 결과
항목설명
사용자검사를 실시한 학습자명
진단명어떤 진단 테스트인지
점수획득 점수
레벨우수/보통/노력 등 자동 분류
CSV우측 상단 "CSV 내보내기" 버튼
CHAPTER 6
사용자 관리 & 메모
가입한 사용자를 관리하고, 학습자별 내부 메모를 남깁니다.
운영 > 사용자 관리
🔍 사용자 검색

이름 또는 아이디로 검색. 유형(학습자/보호자), 상태(활성/비활성) 확인.

✎ 정보 수정

사용자 클릭 > 상세 > "정보 수정" 버튼. 이름, 연락처, 이메일 변경 가능.

🔒 비밀번호 초기화

"비밀번호 초기화" 버튼 > 임시 비밀번호 생성. 화면에 표시된 비밀번호를 사용자에게 전달.

📝 학습자 메모 (신규)

사용자 상세 > "메모" 탭. 관찰/상담/인수인계/일반 카테고리. 고정 기능으로 중요 메모 상단 표시.

👥 할당 콘텐츠

"할당 콘텐츠" 탭에서 배정된 콘텐츠 목록, 상태, 마감일 확인 및 해제.

📈 학습 이력

진단결과, 학습진도, 챌린지결과 탭에서 개별 학습 데이터 확인.

💡 메모 활용법

관찰: 학습 중 관찰한 행동 기록 | 상담: 학부모/학습자 상담 내용 | 인수인계: 담당자 변경 시 전달 사항 | 일반: 기타 메모. 메모는 직원 간 공유용이며, 학부모/학습자에게 보이지 않습니다.

CHAPTER 7
콘텐츠 할당 & 패키지
학습자에게 콘텐츠를 배정하는 방법입니다.

개별/일괄 배정

운영 > 콘텐츠 할당
1

개별 배정

사용자 선택 > 콘텐츠 유형/항목 선택 > 마감일 설정 > 배정

2

일괄 배정

여러 사용자 체크박스 선택 > 콘텐츠 선택 > 한번에 배정

콘텐츠 패키지

콘텐츠 > 패키지 관리
1

패키지 생성

진단/학습/챌린지를 조합하여 패키지로 묶습니다

2

패키지 배정

패키지 선택 > 사용자 선택 > 한번에 여러 콘텐츠 배정

3

추천 규칙 설정

진단 점수 범위별 자동 패키지 배정 규칙을 설정합니다

💡 TIP

콘텐츠가 배정되면 학습자에게 자동 알림이 발송됩니다. 추천 규칙을 설정하면 진단 완료 시 자동으로 적합한 패키지가 배정됩니다.

CHAPTER 8
음성 녹음 관리
학습자가 제출한 음성 녹음을 확인하고 피드백을 남깁니다.
운영 > 음성 녹음 관리
🎧 녹음 목록

기관 소속 학습자들의 전체 녹음 목록을 확인합니다. 프로필별, 콘텐츠 유형별 필터링이 가능합니다.

▶ 재생

녹음을 바로 재생하여 들을 수 있습니다. 재생/일시정지 버튼을 이용합니다.

💬 피드백 작성

"피드백" 버튼 클릭 > 코멘트 입력 > 등록. 학습자에게 자동 알림이 발송됩니다.

👥 학부모 연동

학부모도 자녀 녹음에 피드백을 남길 수 있습니다. 관리자와 학부모의 피드백이 함께 표시됩니다.

💡 TIP

음성 녹음은 학습 콘텐츠와 챌린지 진행 중 학습자가 직접 녹음합니다. 관리자의 따뜻한 피드백 한마디가 학습 동기에 큰 영향을 줍니다.

CHAPTER 9
데이터 내보내기 (CSV)
사용자, 학습 진도, 진단 결과를 CSV 파일로 다운로드합니다.
내보내기 항목 위치 포함 데이터
사용자 목록 운영 > 사용자 관리 > "사용자 목록" 버튼 아이디, 이름, 연락처, 이메일, 유형, 상태, 로그인일, 가입일
학습 진도 운영 > 사용자 관리 > "학습 진도" 버튼 학습자, 콘텐츠명, 유형, 난이도, 상태, 점수, 시도횟수, 시작/최종일
진단 결과 진단 > 진단 결과 > "CSV 내보내기" 버튼 학습자, 진단명, 점수, 레벨, 검사일
💡 파일 형식

CSV (UTF-8 BOM) 형식으로 다운로드됩니다. Excel에서 바로 열어도 한글이 깨지지 않습니다. 파일명에 날짜가 자동 포함됩니다.

⚠ 주의사항

내보내기 시 현재 기관의 데이터만 포함됩니다. MASTER가 "전체"를 선택한 경우 모든 기관 데이터가 포함됩니다.

CHAPTER 10
플랜/기능 관리
기관별 이용 가능한 기능을 플랜 단위로 제어합니다. (MASTER 전용)
1

기능 등록

시스템 > 기능 관리
시스템에서 제어할 기능(key, 이름, 카테고리)을 등록합니다. 예: screening_access, challenge_access 등.

2

플랜 생성

시스템 > 플랜 관리
Basic/Standard/Enterprise 등 플랜을 생성하고, 각 플랜에 포함할 기능을 선택합니다.

3

기관에 플랜 배정

시스템 > 기관 플랜
기관을 선택하고 플랜을 배정합니다. 시작일/만료일을 설정할 수 있습니다.

4

사용자별 오버라이드

사용자 상세 > 기능 탭
특정 사용자에게 플랜과 무관하게 기능을 ON/OFF할 수 있습니다 (세밀한 제어).

CHAPTER 11
FAQ & 문제 해결
Q

관리자 비밀번호를 잊었어요

MASTER가 시스템 > 관리자 관리에서 해당 관리자의 "비밀번호 초기화" 버튼을 클릭하면 임시 비밀번호가 생성됩니다.

Q

사용자가 초대코드를 모르겠다고 해요

운영 > 기관 관리에서 해당 기관의 초대코드를 확인하여 전달합니다. 사용자는 마이페이지에서 나중에 입력할 수 있습니다.

Q

CSV를 Excel에서 열었는데 글씨가 깨져요

UTF-8 BOM이 포함되어 있어 일반적으로 깨지지 않습니다. 구버전 Excel이라면 "데이터 > 텍스트에서"로 가져오기 시 UTF-8을 선택하세요.

Q

특정 ADMIN에게 읽기 전용 권한을 주고 싶어요

MASTER로 시스템 > 권한 관리에서 해당 ADMIN의 관리 권한을 모두 OFF하고, 대시보드/리포트만 ON으로 설정합니다.

Q

담당자가 바뀌면 학습자 정보는 어떻게 인계하나요?

사용자 상세 > 메모 탭에서 "인수인계" 카테고리로 핵심 정보를 기록해두면, 새 담당자가 바로 확인할 수 있습니다.
